# GlobalStatsViewer-Translations
Public hosting of GSV translations
## Guide
The language folder should be titled by the [ISO 639 language code](https://en.wikipedia.org/wiki/List_of_ISO_639_language_codes#Table). For example, English is known as `en`. For a national version of an existing language, use the code followed by `-` and then the [ISO 3166 country code](https://en.wikipedia.org/wiki/List_of_ISO_3166_country_codes#Current_ISO_3166_country_codes). For example, if someone added Brazilian Portugese, it would be `pt-br`. All codes are in lowercase.
